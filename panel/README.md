# Исходники веб-панели

Каталог для разработки и сборки `hy2-admin-panel` (PyInstaller). На ветке **`main`** этого каталога нет — там только установщики и документация.

## Сборка (Linux amd64)

Из корня репозитория на ветке **`dev`**:

```bash
bash tools/build-panel-linux.sh 0.0.4
# артефакт: dist/0.0.4/hy2-admin-panel
```

Публикация: GitHub Releases → `hy2-admin-panel`.
