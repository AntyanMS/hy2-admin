## Изменения

- **Панель → вкладка «Панель»:** честная скорость трафика через сервер (дельта per-user RX/TX из Hysteria trafficStats): скачивание (TX) и отдача (RX).
- Цветные карточки статистики; при наведении — топ-10 активных клиентов по категории (скорость, накопленный трафик, онлайн).
- Исправлено наложение текста подвала на всплывающие списки (z-index).
- **Пользователи:** live-обновление без «перезагрузки» списка — данные через `/api/live` каждые 5 с, без лишней пересортировки DOM; обновляются трафик, онлайн, IP.

## Установка панели

```bash
HY2_PANEL_RELEASE_TAG=v0.0.8 sudo ./install_hy2_admin.sh --auto ...
```

Сборка из исходников (ветка `dev`): `bash tools/build-panel-linux.sh 0.0.8` → `dist/0.0.8/hy2-admin-panel`.
